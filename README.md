## 🛠 Ключевые компетенции

### Бизнес-анализ и проектирование
* **Моделирование процессов:** Описание логики систем в нотациях BPMN и создание Flowcharts (Draw.io/Lucidchart).
* **Продуктовая аналитика:** Формирование состава MVP, анализ рыночных ниш, работа с метриками (LTV, CAC, Retention).
* **Управление требованиями:** Структурирование бизнес-задач и их трансформация в функциональные требования.

### Работа с данными (Data-driven BA)
* **Excel:** Работа со сводными таблицами, сложными формулами (VLOOKUP/INDEX+MATCH), сегментация данных и расчет финансовых моделей.
* **Статистика:** Анализ распределений (медианы, доли), интерпретация данных для принятия бизнес-решений.
